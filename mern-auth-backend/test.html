📚 API Reference

POST Register

Create a new user account

/api/auth/register

Parameters:

  • name: string (required)
  • email: string (required)
  • password: string (required)

POST Login

Authenticate user and get JWT token

/api/auth/login

Parameters:

  • email: string (required)
  • password: string (required)

GET Profile

Get authenticated user's profile

/api/auth/profile

Headers:

  • Authorization: Bearer {token}

GET Health

Check if server is running

/api/health

No parameters required

📖 How to Use

  1. Make sure your backend server is running (npm run dev)
  2. Fill in the Register form with your details and submit
  3. You'll get a JWT token if successful
  4. Use the Login form to authenticate with your email and password
  5. After login, click Get Profile to test the protected route
  6. See the API responses in real-time below each form
  7. Click Logout to clear your session
